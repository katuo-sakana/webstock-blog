# 開発規則

## コミットメッセージ

`git commit -m "prefix(addなど) 変更内容 #issue番号"`

■prefix
fix：バグ修正
add：新規（ファイル）機能追加
update：機能修正（バグではない）
remove：削除（ファイル）
